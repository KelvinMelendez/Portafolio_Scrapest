ABOUT THE PROJECT
This is a project that scrap relevent information from the page https://www.fincaraiz.com.do. This page is about selling living places in Argentina. We can find information such as: price, place, weather, #rooms, #bathrooms and so on. With this tool we can create Scrapping Orders in order to scrap information about living places relevant for you.

If you are interested in the development process of this project, please check the github repository here: Github
